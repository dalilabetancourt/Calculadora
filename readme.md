# Calculadora Web

Este proyecto es una **calculadora web interactiva** desarrollada con **HTML, CSS y JavaScript**.  
Permite realizar operaciones matemáticas básicas directamente desde el navegador.

El proyecto fue creado como práctica para aprender manipulación del **DOM**, eventos en JavaScript y estructura básica de una aplicación web.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

---

## Funcionalidades

La calculadora permite:

- Sumar
- Restar
- Multiplicar
- Dividir
- Calcular porcentaje
- Borrar un número (DE)
- Reiniciar la operación (AC)
